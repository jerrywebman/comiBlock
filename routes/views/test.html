Welcome Bro, These are the existing API Endpoints Base Route: https://comiblock.herokuapp.com/
Route Action Request.body/params
SIGNUP, LOGIN, LOGOUT, CONFIRM EMAIL, RECOVER ACCOUNT & UPDATE ADDRESS
GET-/logout Logout -
POST-/api/signup Signup - req.body.phone, req.body.fullname, req.body.email, req.body.occupation, req.body.dateOfBirth, req.body.password

By default verified, accountVerified and addressVerified state are set to false, and the user is sent an OTP to confirm email address.
POST-/api/confirm_email confirm Email address req.body.email, req.body.verifyCode

Verifies the OTP sent to the user email address with what is stored in the DB and updates the user verified status to true if correct. after account creation. the user should be mandated to confirm email address before the have access to the app
POST-/api/login Authentication/login req.body.email, req.body.password

The user must have verified their email address before they can login to the app
PATCH-/api/add_address To add address req.body.userStreet, req.body.userCountry ,req.body.userCity. user must be logged in to access this route
PATCH-/api/change_password_auth Update Password for logged in user req.body.newPassword ,req.body.confirmPassword. user must be logged in to change_password with this route
PATCH-/api/change_password Update Password for users who lost access to their accounts. req.body.email, req.body.newPassword ,req.body.confirmPassword. no login required to change_password with this route
POST-/api/recover_account Recover Account req.body.email

Sends a new OTP to the user email address
GET-/api/getinfo Response with auth user details res.json({});

Portfolio
POST-/api/portfolio/create Creates comic block Portfolio name: req.body.name, intro: req.body.intro, briefing: req.body.briefing, strategySnapshot: req.body.strategySnapshot, holdings: [req.body.btc, req.body.eth, req.body.usdt], portfolioStrategy: req.body.portfolioStrategy, portfolioType: req.body.portfolioType, subscription: req.body.subscription, subscriptionAmount: req.body.subscriptionAmount,
GET-/api/portfolio Responds with all portfolio res.json([]);
GET-/api/portfolio/:id GETS a portfolio req.params.id, res.json({});
DELETE-/api/portfolio/:id DEL a portfolio req.params.id
